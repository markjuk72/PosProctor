Gmail Setup Instructions
  1. Enable 2-Factor Authentication on your Google account
  2. Go to Google App Passwords
  3. Generate a new App Password for "POSProctor"
  4. Use this 16-character password in the Password field above
  5. Set SMTP Host to: smtp.gmail.com:587
Security Note: Never use your actual Google password. Always use App Passwords for applications.
Test Configuration

  • Checking Grafana logs for SMTP errors
  • Verifying emails are received at the alert recipient address
